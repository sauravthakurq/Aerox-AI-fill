# Security Policy

## Supported Versions

Currently, only the `main` branch of this repository receives security updates. If you find a security issue, please report it immediately rather than waiting for a specific release to confirm the fix.

| Version  | Supported          |
| -------  | ------------------ |
| `main`   | :white_check_mark: |
| Older    | :x:                |

## Reporting a Vulnerability

Your security is our absolute priority. We appreciate all researchers, developers, and users who take the time to disclose any vulnerabilities they encounter to help protect the broader open-source ecosystem.

If you believe you have found a security vulnerability in **Aerox AI Fill**, please report it to us immediately via **Responsible Disclosure**. 

**DO NOT post vulnerability details in public issues or pull requests.**

### How to Contact Us:
To report a vulnerability, please email **Saurav Thakur** securely. 
* Wait for an acknowledgment (usually within 24-48 hours).
* Provide detailed steps that a developer could use to reproduce the security exploit.

If confirmed, the security patch will be rapidly merged and deployed, and an advisory will be published thanking you for your responsible disclosure.
